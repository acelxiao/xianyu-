## 🔧 无法访问Web页面的解决方案

### 方案一：使用一键启动脚本（推荐）

1. **双击运行**：`一键启动系统.bat`
2. **等待3秒**：系统自动启动并打开浏览器
3. **确认访问**：浏览器会自动打开 http://127.0.0.1:5000

### 方案二：手动启动

1. **打开命令提示符**：
   - 按 `Win + R`，输入 `cmd`，回车

2. **切换到项目目录**：
   ```cmd
   cd "C:\Users\Administrator\Desktop\闲鱼"
   ```

3. **启动Web服务**：
   ```cmd
   python web_app.py
   ```

4. **确认启动成功**：
   - 看到 "Please visit: http://127.0.0.1:5000"
   - 看到 "Running on http://127.0.0.1:5000"

5. **打开浏览器**：
   - 手动打开浏览器（Chrome、Edge、Firefox等）
   - 输入地址：`http://127.0.0.1:5000`

### 方案三：使用不同的地址

如果 `127.0.0.1:5000` 无法访问，请尝试：

1. **localhost 地址**：
   ```
   http://localhost:5000
   ```

2. **内网地址**（如果显示）：
   ```
   http://192.168.1.14:5000
   ```

## 🚨 常见问题及解决方法

### Q1: 提示"网页无法访问"

**可能原因：**
- Web服务未启动
- 端口被占用
- 防火墙阻止

**解决方法：**
1. 确认命令行窗口显示 "Running on http://127.0.0.1:5000"
2. 关闭其他可能占用5000端口的程序
3. 临时关闭防火墙测试

### Q2: 提示"此网站无法建立安全连接"

**解决方法：**
- 使用 `http://` 而不是 `https://`
- 确保地址是：`http://127.0.0.1:5000`

### Q3: 浏览器显示"ERR_CONNECTION_REFUSED"

**解决方法：**
1. 检查Web服务是否在运行（查看命令行窗口）
2. 重新启动服务：关闭命令行，再次运行启动脚本
3. 确认Python和Flask已正确安装

### Q4: 页面显示乱码

**解决方法：**
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 使用Chrome或Edge浏览器
