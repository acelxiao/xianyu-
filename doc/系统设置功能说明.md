## 🍪 Cookie管理详解

### ✨ 主要功能
1. **在线更新Cookie**
   - 直接在文本框中粘贴新的Cookie
   - 点击"更新Cookie"按钮即可生效
   - 自动应用到爬虫系统

2. **Cookie有效性测试**
   - 点击"测试Cookie"按钮
   - 自动验证Cookie是否有效
   - 显示测试结果和详细信息

3. **Cookie备份**
   - 一键备份当前Cookie到本地文件
   - 自动命名包含日期时间
   - 防止Cookie丢失

4. **获取指南**
   - 详细的Cookie获取教程
   - 步骤说明和注意事项
   - 安全使用建议

### 🔄 更新Cookie步骤
1. 获取新的闲鱼Cookie
2. 在Cookie管理区域粘贴Cookie字符串
3. 点击"更新Cookie"按钮
4. 等待更新成功提示
5. Cookie自动应用到爬虫系统
